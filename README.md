Apache Solr Kubernetes Operator based on Operator framework SDK.

|-------|---------------|-------------------|

| Level 2 | Seamless Upgrades | :white_check_mark: |
| Level 3 | Full Lifecycle    | :white_check_mark: |
| Level 4 | Deep Insights     | :white_check_mark: |
| Level 5 | Auto Pilot        | :white_check_mark: |

## Description
The Solr Operator manages Solr clusters deployed to Kubernetes 
and automates tasks related to operating a Solr cluster. It provides 
a full management life-cycle for Solr, including deployment, upgrades, 
scaling, and configuration changes.
